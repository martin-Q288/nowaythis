1. 소재 입력

※ 공개 리더 프록시(r.jina.ai)를 경유합니다. 일부 사이트는 차단될 수 있으며, 그 경우 아래에 본문을 직접 붙여넣으세요.
🇰🇷한국 · 🇺🇸미국 · 🇩🇪독일 · 🇯🇵일본 · 🇧🇷브라질 · 🇪🇸스페인어권
캡션 구조: 훅 → 구체적 실화 사례 → 의학 배경 설명 → 전문가 권고 → 권위 출처 인용(예: The Sun, ○○학회) → 팔로우 CTA. 니치 선택 시 발굴 소스도 의학 전용으로 전환됩니다.
사진 1장에 강한 후킹 헤드라인을 얹고, 본문은 캡션에 넣는 방식. 사진 1장이라 릴스(음원)로 그대로 쓰기 좋습니다.

2. 표지 이미지 & 스타일

클릭 또는 드래그해서 이미지 업로드
※ 무료 라이선스(Pexels/Unsplash) 또는 직접 촬영·AI 생성 이미지 사용
G마켓 산스 = 이슈·바이럴 카드의 대표 서체
원본성 안내 — 이 도구는 기사 내용(사실)을 AI가 새 문장으로 재작성합니다. 타인의 게시물 캡처를 그대로 재업로드하는 용도로 쓰지 마세요. 메타 원본 콘텐츠 정책 위반으로 계정 제재 대상이 됩니다. 이미지는 라이선스가 확보된 것만 사용하세요.

3. 결과 미리보기 & 복사/다운로드

소재를 입력하고 [카드뉴스 생성]을 누르면 미리보기가 나타납니다.
카드 이미지를 클릭하면 클립보드에 복사됩니다.
이미지 클릭 = 클립보드 복사 (인스타 웹/PC앱에 바로 붙여넣기 가능) · ⬇ = 파일 저장. 모바일 업로드는 카카오톡 "나에게 보내기"로 파일을 옮기면 편합니다. 생성 비용: 회당 약 $0.01~0.03.

4. 릴스(Reel) 만들기 — 방금 만든 카드뉴스를 세로 영상으로 이어붙여 비팔로워에게 도달

왜 릴스인가 — 카드뉴스(캐러셀)는 주로 이미 나를 팔로우한 사람에게 도달합니다. 새 유입은 릴스가 만듭니다 (릴스탭·탐색탭 노출 → 프로필 방문 → 팔로우 → 그때부터 카드뉴스가 그들에게 도달). 아래에서 현재 언어 탭의 카드들을 9:16 세로 영상으로 자동 변환합니다.
음원 팁 (꼭 읽으세요) — 인스타 인기 음원은 앱에서 직접 올릴 때만 붙일 수 있고, 릴스 도달에 가장 큰 영향을 줍니다. 최대 도달을 원하면 [MP4 저장] → 인스타 앱에서 릴스로 올리며 인기 음원 선택을 권장합니다. 아래 "배경 음원"은 저작권 free 음원을 영상에 직접 입힐 때만 사용하세요 (API 자동 게시용).
무료 음원: YouTube 오디오 보관함 · Pixabay Music · Uppbeat. ⚠ 인스타 인기 음원은 여기에 넣지 말고 앱에서 올릴 때 선택하세요 (여기 넣으면 저작권 제재 위험 + 인기 음원 도달 부스트 못 받음).
👍 권장 경로 — 저장 후 인스타 앱에서 릴스로 올리며 인기 음원 선택 = 도달 최대화
API 게시는 동영상 호스팅(Cloudinary) 필요 · 인기 음원 불가

인스타그램 계정 (언어별 발행)

각 언어는 여기서 매핑한 계정으로 발행됩니다 (🇰🇷한국어 → 한국 계정, 🇺🇸영어 → 영어 계정 …). 계정마다 비즈니스/크리에이터 전환 + 액세스 토큰이 필요해요. 아래 이미지 호스트·소스 설정은 모든 계정 공용입니다.

공용 — 이미지 호스팅 & 소스

ImgBB: imgbb.com 가입 → api.imgbb.com 접속 → "Get API key" → 키 복사해서 위에 붙여넣기. 이게 전부입니다.
Pexels: pexels.com/api 접속 → 가입 → "Your API Key" 복사. 무료·무제한에 가깝고, 모든 사진이 상업적 사용 가능 라이선스라 수익화에 안전합니다.
Cloudinary: Settings → Upload → "Add upload preset" → Signing Mode를 Unsigned로 저장 → preset 이름 입력.
