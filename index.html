0. 오늘의 소재 자동 발굴 — 해외 뉴스 RSS를 기준 점수로 랭킹

이 페이지를 열면 하루 1회 자동으로 크롤링합니다. 실패 시 F12 콘솔에 상세 원인이 기록됩니다.

1. 소재 입력

※ 공개 리더 프록시(r.jina.ai)를 경유합니다. 일부 사이트는 차단될 수 있으며, 그 경우 아래에 본문을 직접 붙여넣으세요.

2. 표지 이미지 & 스타일

클릭 또는 드래그해서 이미지 업로드
※ 무료 라이선스(Pexels/Unsplash) 또는 직접 촬영·AI 생성 이미지 사용
원본성 안내 — 이 도구는 기사 내용(사실)을 AI가 새 문장으로 재작성합니다. 타인의 게시물 캡처를 그대로 재업로드하는 용도로 쓰지 마세요. 메타 원본 콘텐츠 정책 위반으로 계정 제재 대상이 됩니다. 이미지는 라이선스가 확보된 것만 사용하세요.

3. 결과 미리보기 & 복사/다운로드

소재를 입력하고 [카드뉴스 생성]을 누르면 미리보기가 나타납니다.
카드 이미지를 클릭하면 클립보드에 복사됩니다.
이미지 클릭 = 클립보드 복사 (인스타 웹/PC앱에 바로 붙여넣기 가능) · ⬇ = 파일 저장. 모바일 업로드는 카카오톡 "나에게 보내기"로 파일을 옮기면 편합니다. 생성 비용: 회당 약 $0.01~0.03.

API 설정

키는 이 브라우저의 localStorage에만 저장됩니다 (Anthropic API 호출 외 외부 전송 없음).

인스타그램 게시 연동 (선택)

ImgBB: imgbb.com 가입 → api.imgbb.com 접속 → "Get API key" → 키 복사해서 위에 붙여넣기. 이게 전부입니다.
Pexels: pexels.com/api 접속 → 가입 → "Your API Key" 복사. 무료·무제한에 가깝고, 모든 사진이 상업적 사용 가능 라이선스라 수익화에 안전합니다.
Cloudinary: Settings → Upload → "Add upload preset" → Signing Mode를 Unsigned로 저장 → preset 이름 입력.
